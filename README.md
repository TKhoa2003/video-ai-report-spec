# Video AI Report Specification

This repository defines:
- A strict instruction prompt for LLM-based video analysis
- A unified person description schema to ensure consistency and traceability

## Files
- instruction-prompt.txt  
- Schema_PersonDescription.json  
- examples/full_report_example.txt  

## Usage
These files are intended to be loaded directly into LLM pipelines
for surveillance video analysis and report generation.
